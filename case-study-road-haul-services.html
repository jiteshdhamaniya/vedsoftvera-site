About Road Haul Services

Road Haulage Services Ltd is a Haulage company and freight forwarder for the UK and Europe. They do have their own vehicles and partner with others to deliver great freight forwarder service and same-day delivery service.

This is what their website says about themselves.

Road Haulage Services Ltd aim to provide value for money in a rather unsexy element of your business; transport. We have a fleet of our own vehicles and partner with agents on the continent to deliver great value for money for palletised shipping, full and part loads as well as same day delivery.

We put a lot of effort into customer care and as a result we have built a loyal customer base of UK Manufacturers and eCommerce customer selling goods into Europe.

01 Problem

Adam is a Sole-Entrepreneur who does everything on his own. As business started to expand, there were more queries and it was very Hectic job for Adam to handle all these queries in a timely manner.

To make things work Adam was using some automation using Zapier to manage queries from the contact form to Gmail docs but obviously, it was not enough and it became very hard for him to do searching other kinds of stuff even for a simple query. He wanted something which can –

  • Help him to manage queries in a more effective way
  • Easily Find Queries by id or any other detail
  • Easily able to find a Query’s contact details
  • Easily Able to locate a Contact’s all jobs and queries.
  • Make Invoicing Easy
  • And Anything else which can make his life better.

So Adam wanted something which is simple and just does the Job. He contacted us back in March 2019 and with small discussion, we got the idea and started from scratch.

02 Solution

The Idea was to create a Simple Software which does the thing and help Adam to improve his process and save his life. He should be able to manage everything right from there and should save him click as much as we can. So we set these objectives for us –

  • Show all Queries in one Place
  • Manage Emails right from there – View | Draft | Send Emails
  • Extensive Search which can search for contact, Jobs, Companies.
  • Create a Company profile which can be managed to handle contacts
  • Customer Quote from backend for existing customers.
  • Notes to manage Jobs and Contact references.
  • Integration with Existing site which is in WordPress.
  • Merge Contact Functions to manage Duplicate Contacts.
  • Show Reports and Analytics.

Technologies

  • Laravel
  • jQuery
  • Select.js
  • Chart.js for Charts.
  • Bootstrap
  • HTML/CSS

As we set down and finalized with Pre-development, we started out its development. We used Laravel for a backend which is one of our favorites PHP Framework, it’s robust quick and comes with ease for development. For Design, we used one of Bootstrap based Template to make it mobile responsive so he can handle queries even whenever and wherever he wants.

We wanted when he uses the system it should increase his productivity, so for every task, we really did testing that how many clicks one task needed, and we worked to make it as minimum as it could be.

Centralized System

We created a centralized system where all information related to a Job is right there and he need not have to change screen for doing its things, such as sending an email or finding contact details or jobs related to same contact or even adding a new contact to this company. It was fun for us too as we were seeing on a daily basis how much we were making him happy with surprise additions he was not able to visualize.

Emails

To make using the system fun we wanted to him to see related emails and Create send emails right from there. He uses Paid Gmail for his emails so we use Gmail API to make everything this possible for him.

Search Box

We also wanted to make sure he is able to search everything right from one box, so we made this searching box so enrich that he is not able to search for any contact, company or job right from the top search box.

Migration and Merge Contact.

So currently there was no CRM kind of thing, which can create relationships between customers and their jobs, we created that system but now we had to do it for existing data too and This is was very important as he was using a system before us which was kinda making his work a bit efficient so we need some migration technique for data he would use further. Hence we created a merged contact which merged everything of two contacts and now he can use which contact he wants to keep and copy all jobs and everything in that.
